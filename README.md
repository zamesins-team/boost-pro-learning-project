# ПроСервис — учебный проект Boost Pro

Базовый шаблон веб-приложения для онлайн-записи на услуги. Сделан как **стартовый проект для студентов Boost Pro**, у которых нет своего проекта: репозиторий клонируется, и на воркшопах поверх него прикручиваются API, MCP и AI-агенты.

Домен — абстрактный «сервис записи на услуги». В seed-данных для наглядности взят салон (маникюр, стрижка, солярий…), но движок обобщённый: **услуга + мастер + слот + запись**. Чтобы переодеть под свой проект (автосервис, репетитор, коворкинг, клиника) — достаточно поменять seed.

## Что внутри

- **Публичная запись** (`/book`): услуга → мастер → дата и время → контакты.
- **Минимальная админка** (`/admin`): список записей, отмена, управление услугами. Вход по паролю из `.env`.
- **REST API** (`/api/*`): услуги, мастера, свободные слоты, создание записи.

## Стек

Next.js (App Router) · TypeScript · Tailwind CSS · PostgreSQL · Prisma — тот же стек, что в основной программе Boost.

## Как запустить локально

Нужны **Node.js 20+** и **Docker** (для базы данных).

```bash
cd project
cp .env.example .env          # при необходимости поправьте пароли
docker compose up -d          # PostgreSQL на localhost:5432
npm install
npm run db:push               # создаст таблицы
npm run db:seed               # зальёт пример данных (салон)
npm run dev                   # http://localhost:3000
```

Полезные команды:

- `npm run db:reset` — пересоздать БД и перезалить seed.
- `npm run db:studio` — Prisma Studio (просмотр/правка данных в браузере).

Дальше на воркшопах поверх этого проекта добавляются: внешний API (W2), MCP к этой же базе (W3), предсказуемый ИИ и бот (W5).

## Структура

```
project/
├── docker-compose.yml      # PostgreSQL для локальной разработки
├── prisma/
│   ├── schema.prisma       # модель данных
│   └── seed.ts             # пример данных (салон)
└── src/
    ├── app/
    │   ├── page.tsx        # главная (список услуг)
    │   ├── book/           # онлайн-запись
    │   ├── admin/          # админка (пароль из .env)
    │   └── api/            # REST: services, masters, availability, bookings
    └── lib/
        ├── prisma.ts       # клиент Prisma
        ├── availability.ts # расчёт свободных слотов
        └── format.ts       # форматирование цен/времени
```
